# Search-algorithms
2 codes written in D which implement the BFS and the DFS algorithms.
There is an image file which shows an illustration of the graph.
bfs.d is the source code with the BFS algorithm, compile it with "dmd bfs.d -m64 -O -g".
dfs.d is the source code with the DFS algorithm, compile it with "dmd dfs.d -m64 -O -g".
